# 🚀 Multi-Agent 协作智能任务系统

> 基于多 Agent 协作的智能任务处理系统，支持任务拆解、执行与评估的闭环优化流程。

---

## 📌 项目简介

本项目实现了一个轻量级多 Agent 协作框架，用于解决复杂任务的一步到位生成效果不稳定的问题。  
系统通过引入多个功能分工明确的 Agent，将任务处理过程拆分为多个阶段，从而提升结果的逻辑性与准确性。

相比传统单轮生成方式，本系统能够通过“规划 → 执行 → 评估”的链式推理流程，实现更稳定、更可控的任务完成效果。

---

## 🧠 系统架构

系统由三个核心 Agent 构成：

- **🧠 Planner（任务规划）**
  - 将复杂任务拆解为结构化步骤
  - 提供清晰执行路径

- **⚙️ Executor（任务执行）**
  - 根据 Planner 输出逐步完成任务
  - 生成具体执行结果

- **✅ Critic（结果评估）**
  - 判断任务是否完成
  - 提供改进建议
  - 支持后续优化扩展

---
